Publish/Subscribe реализация - передача разных получателям одинаковых сообщений.
Тут используем Exchange с типом Fanout, который будет рассылать консамерам сообщения
Не указываем наименование очереди, т.к. это важно только для разделения сообщений
Без запущенных консамеров, сообщения не будут храниться в exchanges (обязательно должна быть очередь, куда складывать сообщения)
